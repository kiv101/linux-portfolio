- Сортирует процессы по CPU и памяти
- Работает без прав root (использует доступные логи)

### 1. sysinfo.sh
Скрипт для быстрой диагностики системы. Показывает:
- uptime, загрузку CPU
- память и диски
- топ процессов
- ошибки из логов
