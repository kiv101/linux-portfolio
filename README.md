### 2. backup.sh
Скрипт для автоматического резервного копирования папок. Создаёт сжатый архив с меткой времени.

**Что умеет:**
- Архивирует заданную папку в формат `.tar.gz`
- Добавляет в имя файла точную дату и время (например, `backup_scripts_2026-05-12_15-30-00.tar.gz`)
- Кладет архивы в отдельную папку `~/backups`
- Показывает размер созданного бэкапа и список всех сохраненных копий

### 3. check-disk.sh
Скрипт для мониторинга свободного места на дисках. Позволяет заранее заметить нехватку места.

**Что умеет:**
- Проверяет все реальные диски (игнорирует временные файловые системы)
- Показывает удобную таблицу: размер, использовано, доступно, проценты
- Отправляет предупреждение, если свободного места осталось меньше заданного порога (например, 10%)
- Возвращает код ошибки, чтобы его можно было использовать в системах мониторинга или cron

[Посмотреть код скрипта](03_monitoring/check-disk.sh)
